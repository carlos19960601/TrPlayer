**TrPlayer** 支持转录，利用 llm 对转录文本进行翻译的视频播放器。

![TrPlayer](./snapshots/home.png)

### 功能

- Whisper 模型管理，内置模型下载
- 支持播放视频和字幕
- 转录视频
- 翻译字幕
- 导出 ass 字幕文件和视频

### 技术栈

- Electron
- Shadcn UI
- ffmpeg
- tailwind
- whisper.cpp

大模型提供商

- openrouter
- ollama
- siliconflow
- openai
